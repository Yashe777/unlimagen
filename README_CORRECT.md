# Unlimagen

AI Image Generator - Live at https://unlimagen.com

## Quick Start

```bash
pip install -r requirements.txt
python app.py
```

## Environment Variables

Create `.env` file:
```
HUGGINGFACE_TOKEN=your_token
```

## Deployment

Deployed on Render.com - Auto-deploys from main branch

## Tech Stack

- Python Flask
- Hugging Face API
- JSON database
- Session-based auth

## Notes

- Free plan: 10 images/day
- Basic plan: 50 images/day ($9.99/mo)
- Pro plan: Unlimited ($29.99/mo)
- Google AdSense integrated
- Mobile responsive design
